# FILE-INTEGRITY-MONITORING-USING-WAZUH

Project Overview
This project demonstrates the implementation of File Integrity Monitoring (FIM) using the open-source security platform Wazuh on an Ubuntu-based environment.
The objective of this project is to detect, monitor, and alert unauthorized changes in critical system files in real-time. This is a core capability of Security Operations Center (SOC) environments and is essential for detecting insider threats, malware activity, and unauthorized access.
